## Introduction

Gecko is a faker of website named [**ZhiHu**](http://www.zhihu.com), just for fun and study.

* jQuery 1.11.3
* Bootstrap 3.3.5
* UEditor 1.4.3
* jquery.validate 1.9.0
* html2canvas-0.5.0-beta4
* font-awesome-4.7.0
* jsp / servlet
